HERO-ML

This is the website of HERO-ML, an array language on very high level, which is intended for specifying data parallel algorithms in a concise and platform-independent way where all the inherent data parallelism is easy to identify. The goal is to provide support for the software development for heterogeneous systems with different kinds of parallel hardware, where programs tend to be very platform-specific and difficult to develop.
