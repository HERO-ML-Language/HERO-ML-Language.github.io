2. HERO-ML overview

HERO-ML is an imperative, fully sequential language extended with a data parallel array data type. Thus, all the parallelism is expressed through operations over these arrays. The sequential part of the language is deliberately kept simple, since its main purpose is to demonstrate the principles of very high-level data parallel programming rather than providing a full-fledged production language. It is a standard WHILE language [6], and the parallel part adds:

  • a type for so-called abstract arrays (see Section 4),
  • a set of array expressions,
  • a statement to evaluate such expressions and bind the resulting array to a program variable, and
  • a masked concurrent assignment, where all the elements in an existing array that fulfil some condition are updated concurrently, in-place.
